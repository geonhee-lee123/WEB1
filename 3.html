HyperText MarkUp Language (HTML) is the standard markup language for creating
web pages and web applications. Web browers receive HTML documents from a web
server of from local storage and render them into musltimedia web pages. HTML describs the
structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. width
HTML constructs, images and other objects, such as interactive forms, may be denoting structural
